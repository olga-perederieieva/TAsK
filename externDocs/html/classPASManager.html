void PASManager::createNewPAS ( DAGraphTapas *  dag,
StarLink *  expLink,
int  mergingNodeIndex 
)

Checks if PAS with a specified expensive link expLink already exists. If it exists, method adds current bush to the set of relevant origins of the existing PAS and checks if current PAS is effective. If PAS does not exists the method creates a new PAS (if it is possible) and checks if it is effective. If created or existing PAS is not effective, one more PAS is created that is effective.

Note
Branch shift is not implemented.
